☁️ Como funciona

Este serviço roda 24/7 na nuvem (Render.com).


Pipeline automático:

① AliExpress API → produtos do dia

② Claude AI → título SEO + descrição + hashtags + texto Telegram

③ Pinterest API → Pin com imagem do produto

④ Telegram Bot → mensagem com foto para o canal


Horários: baseados em dados de 2M+ posts do público brasileiro (BRT)


SEO de imagem:

• Alt text detalhado para busca visual

• Título com keyword principal + benefício

• 8-10 hashtags relevantes em português


Para configurar: adicione as variáveis de ambiente no Render.com dashboard
